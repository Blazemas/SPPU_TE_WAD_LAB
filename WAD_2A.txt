######### ALl Commands are in GIT Bash #############

Step 1: Create a GitHub Account

Step 2: Install Git


Step 3: Set Up Git


->git config --global user.name "Your Name"
->git config --global user.email "youremail@example.com"

Step 4: Create a New Repository


Step 5: Clone the Repository

->git clone https://github.com/your-username/my-first-repo.git

Step 6: Create a Simple README File

->touch README.md (Runs of Git Bash only)

# My First Repository

This is my first repository on GitHub. It's just a simple example.
Save the file.

Step 7: Add, Commit, and Push Your Changes

->git add README.md
->git commit -m "Add README file"
->git push origin master 
->git push origin main
